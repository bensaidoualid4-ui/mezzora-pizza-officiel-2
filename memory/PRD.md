# Mezzora Pizza - PRD

## Problem Statement
Site vitrine élégant pour "Mezzora Pizza" à Rueil-Malmaison (92500).

## Design Direction
Inspiré de restaurantpassionne.com — noir, élégant, simple, serif italique, doré.

## Palette
- Background: #0B0B0B (noir)
- Text: #F5F0E8 (crème)
- Muted: #A89F91 (crème atténué)
- Accent: #C8A96E (doré)
- CTA: #C41E3A (rouge)
- Fonts: Cormorant Garamond (serif) + Inter (sans)

## Key Files
- Hero.js, BestSellers.js, AboutSection.js, RubricsSection.js
- MenuSection.js, FormulesMidi.js, OffresSection.js
- ReviewsCarousel.js, ContactForm.js, Footer.js, Header.js
